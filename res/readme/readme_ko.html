사용자 체험 프로그램에 참여합니다

제품의 품질을 향상 및 고객님의 이용 현황을 파악하기 위하여 저희는 사용자의 사용 정보를 수집합니다. 이 정보는 식별 또는 고객님과 연락하고자 하는 것이 아닙니다. 고객님은 "도움말(Help)"메뉴에서 이 기능을 취소할 수 있습니다.
1 장치 PLE.
2 소프트웨어 시작/종료 시간.
3 스캔 설정:
   스캔 모드, 컬러, 정확도.
4 데이터 정보:
   point count,length,width,height,save file name.
5 시스템 정보:
   운용체제, 언어, 해상도, 소프트웨어 설치 경로, 설치디스크 사이즈.
6 하드웨어 정보:
   CPU Type, RAM, GPU Type.
