Kullanıcı Deneyimi Programı

Ürünlerimizi geliştirmek ve bunları nasıl kullandığınızı anlamak için, kişisel olmayan ürün kullanımına ilişkin bilgileri toplarız. Bu bilgiler, sizi tanımlamak veya sizinle iletişime geçmek için kullanılmaz. Bunu yardım menüsünden kapatabilirsiniz.
1. Cihazınızın PLE'si.
2. Yazılımı başlatma ve kapatma zamanı.
3. tarama ayarı:
   tarama modu, doku(renk), doğrulukg.
4. data bilgisi:
   point count,length,width,height,save file name.
5. sistem bilgisi:
   işletim sistemi, dil, çözünürlük, yazılım yükleme yolu, kurulum diski boyutu.
6. hizmet bilgisi:
   CPU tipi, RAM, GPU tipi.
