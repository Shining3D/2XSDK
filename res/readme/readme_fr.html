Participer à l'amélioration du logiciel

Pour améliorer la qualité de ce produit et mieux en comprendre l'utilisation, nous collectons les retours des utilisateurs. Ces données ne peuvent être utilisées pour vous dentifier ou vous contacter. Vous pouvez désactiver cette option dans le menu d'Aide
1 PLE de votre appareil.
2 Lancer et fermer le programme.
3 Paramètres de scan:
   mode, texture, précision.
4 Info sur les données:
   point count,length,width,height,save file name.
5 Info sur le système:
   système d'exploitation, language, résolution, chemin d'installation logiciel, définir l'espace disque.
6 Info logiciel:
   type de CPU, RAM, type de GPU.
