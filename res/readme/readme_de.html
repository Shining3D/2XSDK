Nehmen Sie am User Erfahrungsprogramm teil

Um die Produktqualität zu verbessern und die Anwendung zu verstehen, sammlen wir Feedback von Anwendern. Diese Daten werden nicht dazu verwendet um Sie zu indentifizieren oder Sie zu kontaktieren. Sie können diese Funktion im Hilfe Menü abschalten.
1 Ihr Gerät PLE.
2 Die Zeit die Software zu starten und zu schliessen.
3 Scanning Setting:
   Scan Mode, Texture, Genauigkeit.
4 Daten-Information:
   point count,length,width,height,save file name.
5 System-Information:
   Operating System, Sprache, Auflösung, Software Installationspfad, Einstellung Festplattengröße.
6 Hardware-Info:
   CPU Typ, RAM, GPU Typ.
