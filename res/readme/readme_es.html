Participe en el programa de experiencia del usuario

Con el fin de mejorar nuestros productos y entender cómose usan, recopilamos información de uso de productos no personales. Esta información no se utiliza para identificarle o contactarlo. Puede desactivar esta opción en el menú de ayuda.
1. El PLE de su dispositivo.
2. El momento para abrir y cerrar el software.
3. Parámetros de Escaneo:
   Modo de Escaneo, textura, precisión.
4. Información de datos:
   cuenta de puntos, longitud, ancho, altura, guardar nombre de archivo.
5. Información del Sistema:
   Sistema Operativo, lenguaje, resolución, ruta de la instalación del software, tamaño del disco de configuración.
6. Información del Hardware:
   Tipo de CPU, RAM, Tipo de GPU.
