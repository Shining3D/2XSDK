Weż udział w programie doświadczony użytkownik

Aby poprawić jakość produktu i ułatwić jego obsługę zbieramu dane od użytkowników. Dane te nie będą użwyane do identyfikacji czy kontaktu z użytkownikiem. Możesz wyłączyć tę opcję w meni pomocy.
1 Twoje urządzenie PLE.
2 czas do uruchomienia i zamknięcia programu.
3 ustawienie skanowania:
   tryb skanowania, tekstura, precyzja.
4 Informcja o danych:
   point count,length,width,height,save file name.
5 Inforamcje o systemie:
   system operacyjny, język, rozdzielczość. Ścieżka instalacji oprogramowania, wielkość dysku na którym zainstalowane jest oprogramowanie.
6 Inforamcja o hardware:
   typ Procesora, RAM, Ty procesora graficznego karty.
